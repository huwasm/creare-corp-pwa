Design Decisions

Q&A

Why 3 cards side-by-side instead of one chart with tabs?

Instant overview. Judges see all 3 commodities at a glance — Buying Zone / Wait / Bearish. No clicks needed. This is the pacemaker.ai pattern: show the answer first, let users drill into detail.

What do the signal metrics mean?

14d trend: % price change over 14 days. DMA trend: days until moving average crossover (buy/sell signal). Volatility: 30-day coefficient of variation. Accuracy (MAPE): mean absolute percentage error of the forecast model over 90 days.

How does the expanded chart work?

Click any card → it expands into a full chart below the cards row. Gold border shows which card is selected. Close button (✕) collapses it. Chart has Line/Area/Candle pills + time range + crosshair tooltip + stats row.

What is the Event Feed?

News table from 30200_news. Columns: Date, Material (icon), Headline, Source (badge), Event Types (gain/loss/flat pills), Sentiment (dot + label), Breakdown (stacked color bar showing sentiment distribution). Filterable by material and date range.

What is the Breakdown bar?

A stacked horizontal bar showing sentiment distribution: green (positive) → yellow (neutral) → orange (slightly negative) → red (negative). Computed from the news summary using keyword-based sentiment scoring. Quick visual scan without reading numbers.

What data feeds each section?

Cards: Latest price from 30100_prices + computed signals (trend, volatility, DMA from price history). Chart: Full price series filtered by date range. Event Feed: 30200_news with sentiment + event type classification added by the Signal Engine.
