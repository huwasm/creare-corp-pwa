Design Decisions

What stays the same between tabs?

Header, sidebar, resize handle, and chat panel are identical. Only the main content area changes. The chat adapts its sticky section — Charts tab shows latest news, Briefing tab shows all 3 recommendations as a summary.

Can the chat talk about the briefing?

Yes. When the Briefing tab is active, the chat's system prompt includes the computed briefing data (signals, scores, recommendations). The user can ask "Why avoid copper?" and get answers grounded in the actual Price Block + Sentiment Block data.

How are the 3 briefing cards computed?

Each card runs the full pipeline: ROC momentum + ARIMA forecast → Signal. FinBERT decay-weighted sentiment → Tier. Signal × Tier → Decision Matrix → Recommendation. Flag from volatility. LLM writes the narrative. All computed on page load.

What changes in chat between tabs?

Charts tab: sticky shows 3 latest news for selected commodity. Chat answers about prices and news. Briefing tab: sticky shows 3 commodity recommendations. Chat answers about the briefing analysis, comparisons between metals, timing advice.
